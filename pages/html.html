Introduction à HTML

L'HTML a été créé par Tim Berners-Lee en 1991 afin de partager des documents scientifiques sur le Web. Aujourd'hui, le langage reste la base de toute page web : il définit la structure du contenu et permet de hiérarchiser l'information.

Balises essentielles

Les balises structurent un document HTML. Elles encadrent le contenu et peuvent accepter des attributs pour apporter des précisions.

  • <h1> à <h6> pour les titres hiérarchisés.
  • <p> pour déclarer un paragraphe de texte.
  • <a> pour créer un lien hypertexte.
  • <img> pour afficher une image.
  • <section> pour grouper un ensemble d'éléments liés.

Exemple de structure HTML

            
<!DOCTYPE html>
<html lang="fr">
  <head>
    <meta charset="UTF-8" />
    <title>Exemple de page HTML</title>
  </head>
  <body>
    <h1>Bienvenue sur ma page web</h1>
    <p>Ceci est un paragraphe d'exemple.</p>
    <a href="https://www.example.com">Visitez mon site</a>
  </body>
</html>

Attributs essentiels d'un formulaire

  • method : définit la manière dont les données sont transmises (GET ou POST). POST est privilégié pour envoyer des informations sensibles.
  • action : indique l'URL du script ou de la page qui recevra les données saisies.
  • required : oblige l'utilisateur à renseigner un champ avant l'envoi du formulaire.
  • name : associe un identifiant à chaque champ afin d'exploiter sa valeur côté serveur.
