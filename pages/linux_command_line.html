Commandes Linux indispensables

Ce mémento rassemble les commandes étudiées pendant les séances de SISR. Chaque action est accompagnée d'un exemple concret et d'une description pour faciliter la révision.

Mémento rapide

Tableau de synthèse des commandes Linux
Commande Description Exemple d'utilisation
ls Liste le contenu du répertoire courant. ls -la
cd Change de répertoire. cd /var/www
mkdir Crée un nouveau dossier. mkdir sauvegardes
rm Supprime un fichier ou un dossier. rm fichier.log
cp Copie un fichier ou un répertoire. cp index.html backup/index.html
pwd Affiche le chemin du répertoire courant. pwd

Comprendre la structure de la table

  • <table> encapsule l'ensemble du tableau.
  • <caption> fournit un titre accessible décrivant le contenu.
  • <thead> regroupe l'en-tête et <th> définit les cellules de titre.
  • <tbody> contient les lignes de données composées d'éléments <tr>.
  • <td> représente une cellule standard tandis que scope améliore l'accessibilité en reliant les titres aux cellules de données.
